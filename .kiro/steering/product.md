# Product Overview

This is an ESP-IDF Hello World example project that demonstrates basic ESP32 firmware development. The project creates a simple FreeRTOS application that prints system information and performs a countdown restart.

## Purpose
- Educational example for ESP-IDF development
- Demonstrates basic ESP32 chip information retrieval
- Shows FreeRTOS task management and system restart functionality
- Serves as a minimal template for ESP32 projects

## Target Hardware
Supports all ESP32 family chips including ESP32, ESP32-C2, ESP32-C3, ESP32-C5, ESP32-C6, ESP32-H2, ESP32-S2, ESP32-S3, and Linux host testing.
